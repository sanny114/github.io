🤔第1部：「結局どの生成AIを使えばいいですか？」 (8分)

2種類の生成AIを紹介：

1
検索系AI：「最新情報を調べたいとき」
  • Perplexity、Genspark、Felo
  • 使用例：「今の総理大臣は誰？」
2
汎用系AI：「最新情報の検索以外で使いたいとき」
  • ChatGPT、Claude、Gemini
  • 使用例：「新商品のキャッチコピーを5つ考えて」「悩みを聞いて」

初心者向け推奨：ChatGPT & Perplexity の組み合わせ（ただし、他の生成AIもすぐ使える状態にしておこう）

  • なぜこの組み合わせ？：有名で情報が豊富、検索で解決策が見つけやすい
  • 実体験ストーリー：「ChatGPTが最近のアップデート時に使えなくなって困った時、予備のClaudeがあって助かった」

個人的な使い分け紹介：

「私自身は最近ChatGPT & Perplexityも使いますが、Claude & Feloの組み合わせも多いです」
「各ツールの得意分野がわかってくると、目的に応じて使い分けられるようになります」

課金するべき？：

  • 「最初は無料版で十分スタートできます、制限はあるが有料限定機能が無料で使えるツールも多い」
  • 有料版（ChatGPT Plusなど）の特徴：
    • 回答が早い
    • より正確な回答が得られる
    • 画像生成ができる
    • GPTs（カスタム機能）が使える
  • 課金を検討するタイミング：「自分の使い方が見えてきてから」

🔄第2部：「生成AIを使う習慣が身につきません」 (7分)

「私も最初は『使わなきゃ』と思いつつ忘れてばかりでした」

習慣化の2ステップ：

1
環境を作る：「使いたいと思ったら2秒で使える状態にする」
  • PCでの環境設定：
    • お気に入りに入れておく
    • デスクトップにショートカットを置く
    • デスクトップアプリをインストールする
  • スマホでの環境設定：
    • アプリをトップ画面のすぐ使える場所に置いておく
「生成AIを使うのに10秒以上かかるなら環境の見直しが必要です」
2
トリガー決め：「○○したら生成AIを使う」と決めておく
  • 時間・タイミングのトリガー：
    • 朝起きたら...今の気分を入力して整理してもらう
    • 仕事の開始前に...今日の仕事の最優先事項の進め方を相談する
    • 夕食作りの前に...今日の夕飯のメニューを相談する
  • 状況のトリガー：
    • Google検索したくなったら...（生成AIで）調べる
    • 仕事で手が止まったら...タスクの全体像と進め方を相談する
    • 職場・家族のコミュニケーションに困ったら...とにかく話を聞いてもらう
「トリガーを『Google検索をする前にPerplexityで調べる』と決めただけで、1週間で使用頻度が5倍になりました」

🔍第4部：「思った通りの回答が返ってこないのですが？」 (5分)

「私も最初は『こんなはずじゃなかった』と思う回答ばかりでした」

回答精度を上げる3ステップアプローチ：

1
モデル選択：目的に合ったAIツールを選ぶ
  • 最新情報が必要なら検索系AI（Perplexity等）
  • 文章作成やアイデア出しなら汎用系AI（ChatGPT、Claude等）
「同じ質問をPerplexityとChatGPTで試してみたら全く違う回答が返ってきた時の驚き」
2
インプット（プロンプト設計）：効果的な質問をする
  • 背景・文脈情報を追加：「私はマーケティング部門で働いています。経営層向けの提案資料を作成しています。」
  • 前提条件の共有：「ExcelのVBAは初めてです」「すでに○○と△△は試しました」

WWWWHを意識したプロンプト設計：

  • Who（誰に向けた内容か）：「小学生向けに」「マーケティング初心者向けに」
  • What（何について）：テーマを明確に
  • Why（なぜ必要か）：「提案資料を作るために」など目的を伝える
  • When（いつの情報か）：「最新の」「2024年の傾向を踏まえた」
  • How（どのような形式で）：「箇条書きで」「500文字以内で」
「人に依頼するときと同様に必要な情報を洗い出してから渡しましょう」

プロンプト改善の具体例：

1
ビフォー：「マーケティングについて教えて」
↓
2
アフター：「20代向けInstagramの3ステップを300字で具体例を交えて教えてください。私はアパレルブランドのSNS担当で、フォロワーを増やしたいと思っています。」

🧠第4部：続き

使えるテンプレート紹介：

「{ トピック } について { 対象者 } 向けに説明してください。特に { 重要なポイント} を中心に、{ 文字数 } 程度でまとめてください。具体例も入れてください。」
「私は { あなたの立場 } で、{ 目的 } のために調べています。{ トピック } について知りたいのですが、特に{ 詳しく知りたい点 } を教えてください。私は { 前提知識レベル } です。」
3
アウトプット（期待値調整）：対話で理想の回答に近づける
  • 「AIをパートナーとして捉え、一緒に考えを発展させる」アプローチ
  • 段階的な質問例：
    • 1回目：「ブログ記事の構成案を作って」
    • 2回目：「第2章をもう少し詳しく展開して」
    • 3回目：「ターゲット読者を30代主婦に変更して」

「ちょい足し」プロンプトテクニック：

  • 「小学生にもわかるように」：難しい説明をシンプルに
  • 「ステップバイステップで考えて」：思考過程を見せてもらう
  • 「具体例を3つ追加して」：抽象的な内容を具体化
  • 「表形式にして」：情報を整理する
  • 「今の回答を60点として、100点満点の回答をお願いします」：回答の質を上げる
「『小学生にもわかるように』というプロンプトを足しただけで、AIの回答が劇的に分かりやすくなりました」
